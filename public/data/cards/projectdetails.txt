id: telegram-apollo-lead-finder
overview: A smart automation workflow that interprets text or voice from Telegram and performs Apollo.io lead scraping with AI-generated search queries, delivering structured B2B leads directly into Google Sheets.
problem: Manual lead scraping and list generation was slow, error-prone, and disconnected from real user workflows, reducing sales team productivity.
solution: Built an end-to-end n8n-based workflow that takes natural language input from a Telegram bot, transforms it into optimized Apollo.io search queries using AI, scrapes lead data via Apify, and outputs structured results into Google Sheets.
impact:
  - Time Saved: Significant reduction in manual lead gathering
  - Workflow Efficiency: Seamless integration from Slack to Sheets
  - Data Quality: Consistent output with automated enrichment
features:
  - Telegram native bot for text and voice input
  - AI-based natural language to query translation
  - Apollo.io scraping using Apify
  - Structured data delivery to Google Sheets
  - Error handling and status updates via bot responses
tech:
  frontend: Telegram Bot API
  workflow: n8n
  ai: OpenAI (GPT-4o-mini, Whisper)
  scraping: Apify Apollo.io Scraper
  data_storage: Google Sheets API
deployment: Hosted automation logic with secure API credentials and scalable workflow execution
